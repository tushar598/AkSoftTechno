Learn MERN Stack : Introduction


Mern Stack is a collection of technologies used to build full-stack web applications. It consists of MongoDB, Express.js, React.js, and Node.js. This course will guide you through the fundamentals of each technology, enabling you to create dynamic and responsive web applications.

Chapters

Web Development Fundamentals
  • What is Full-Stack Development?
  • Overview of Frontend, Backend, and Databases
  • Introduction to the MERN Stack
  • Setting up the Development Environment (VS Code, Node, MongoDB, Git)
HTML, CSS & JavaScript Refresher
  • HTML5 Semantic Elements
  • CSS3 Flexbox, Grid, Media Queries
  • JavaScript ES6+ Essentials (let, const, arrow functions, promises, fetch, async/await)
  • DOM Manipulation, Events, and Form Handling
React.js (Frontend)
  • Introduction to React
  • Functional vs. Class Components
  • JSX and Virtual DOM
  • Props and State
  • Event Handling and Forms
  • Conditional Rendering
  • Lists and Keys
  • useState and useEffect Hooks
  • Component Lifecycle (overview)
  • Context API for State Management
  • Routing using react-router-dom
  • React Forms (controlled/uncontrolled)
Node.js (Backend Runtime Environment)
  • Introduction to Node.js
  • Setting up Node.js with npm
  • Modules and require vs import
  • File System, OS, Events
  • Creating a basic HTTP server
  • Introduction to Express.js
Express.js (Backend Framework)
  • Express.js Basics
  • Routing and Middleware
  • Request and Response Objects
  • REST API Basics (CRUD operations)
  • JSON Handling
  • Connecting to MongoDB using Mongoose
  • Validation using Express Validator or JOI
  • Error Handling and Status Codes
  • CORS setup for frontend-backend communication
MongoDB (Database)
  • Introduction to NoSQL and MongoDB
  • Installing and Using MongoDB Compass
  • Creating Databases and Collections
  • CRUD Operations (Create, Read, Update, Delete)
  • Mongoose ODM (Schemas and Models)
  • Relationships (One-to-Many, Many-to-Many)
  • Aggregation and Indexes (basics)
Connecting MERN Stack
  • Connecting React Frontend to Express Backend
  • Calling APIs from React using Axios
  • Working with JSON and Form Data
  • Protected Routes and Role-based Access
  • Building a Full CRUD MERN App
